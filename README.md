# 🎓 Django Student Management System

A web-based Student Management System built using Django that allows efficient handling of student records, attendance, and data management.

---

## 🚀 Features

- 👨‍🎓 Student Registration
- 📋 Manage Student Records (CRUD)
- 📊 Attendance Tracking
- 🔍 Search & Filter Students
- 🔐 Admin Panel (Django Admin)

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Database: SQLite
- Frontend: HTML, CSS
- Version Control: Git & GitHub
